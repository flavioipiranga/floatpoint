O que é o projeto Floatpoint?

É uma implementação do padrão IEEE 754 para representar números em ponto flutuante.

Download e Instalação

Para executar o programa é necessário instalar a biblioteca (responsável por computar o formato conhecido como Big Int), que se encontra no link abaixo:

www.gmp.org

Caso você esteja usando alguma distro do Linux pode usar o comando:

$ sudo aptitude search gmp

Depois de instalar a basta compilar o código e executá-lo da seguinte forma:

$ cd Floatpoint/floatpoint/
$ make 
$ ./executavel

Bugs

Ao rodar numa distro Linux, é possível que a função malloc do C, apresente alguns erros. Isso ocorre pelo fato de que o software foi desenvolvido utilizando um OS X(Mac), no qual o código compila e executa normalmente.
